# FinTracker

**A simple .NET console application for tracking your daily expenses.**

## üìå Overview
FinTracker helps you manage your expenses by allowing you to add, view, filter, and summarize your spending. All data is saved to a JSON file, so your records persist between sessions.

---

## ‚ú® Features
- **Add Expenses**: Log expenses with a name, amount, and category.
- **View All Expenses**: Display a list of all recorded expenses.
- **Filter by Category**: View expenses for a specific category.
- **Summary Statistics**: See total spent, most expensive category, and average expense.
- **Data Persistence**: Expenses are saved to a JSON file for future use.
